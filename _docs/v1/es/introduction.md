# Introducción

Bienvenido al sitio de **documentación multi-idioma**. Este proyecto demuestra el poder de Next.js con **ISR** e **i18n**.

## Características Clave

- Internacionalización (en, es, fr, de)
- Regeneración Estática Incremental (ISR)
- Búsqueda de texto completo
- Referencia de API con Swagger UI
