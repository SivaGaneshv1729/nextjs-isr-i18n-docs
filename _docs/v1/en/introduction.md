# Introduction

Welcome to the **multi-language documentation site**. This project demonstrates the power of Next.js with **ISR** and **i18n**.

## Key Features

- Internationalization (en, es, fr, de)
- Incremental Static Regeneration (ISR)
- Full-text search
- API Reference with Swagger UI
